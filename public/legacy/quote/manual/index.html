1. 시작하기

이 앱은 여행 견적 업무를 한 화면에서 처리하도록 만든 통합 도구입니다.

상단 제목은 견적 계산기_2026 버전이며, 여러 견적 파일을 탭으로 동시에 열어 병행 작업할 수 있습니다.

  1. 상단 파일 탭에서 견적 파일을 열거나 새 파일을 생성합니다.
  2. 고객 정보/메모를 입력합니다.
  3. 견적 그룹에서 계산, 항공, 요금, 포함/불포함, 호텔, 일정표를 작성합니다.
  4. 저장 또는 다른 이름으로 저장으로 파일을 관리합니다.

2. 화면 구성

  • 파일 탭 바: 여러 견적 파일을 탭으로 전환.
  • 상단 버튼: 새창, 저장, 다른 이름 저장, 불러오기, 최근 파일.
  • 좌측: 고객 정보 카드.
  • 가운데: 메모 + DB 불러오기.
  • 우측: url 단축기 임베드 화면(새 창에서 열기 가능).
  • 하단: 견적 그룹 탭/버튼 + 그룹 상세 모듈.

3. 파일 탭/저장 기능

3.1 파일 탭

  • 상단 + 버튼으로 새 견적 파일 탭 생성.
  • 탭 클릭으로 파일 전환.
  • 탭의 x로 파일 탭 닫기.
  • 주황 점은 저장되지 않은 변경사항(Dirty) 표시.

3.2 저장/불러오기

  • 저장(F2): 현재 파일 덮어쓰기.
  • 다른 이름으로 저장(F3): 새 파일로 저장.
  • 불러오기(F4): 기존 파일 열기.
  • 최근 파일(Shift+Y): 최근 파일 빠른 열기.
  • 새창(Shift+N): 같은 앱을 새 창으로 열기.

4. 견적 그룹 작업

4.1 그룹 관리

  • +새 그룹: 새 견적 그룹 추가.
  • 그룹 복사: 현재 그룹 전체 복제.
  • 그룹 삭제: 현재 그룹 삭제.
  • 그룹 탭 이름은 더블클릭으로 수정.

4.2 견적 계산기

  • 계산기 추가/복사 가능.
  • 원가, 상품가, 수익, 수익률 자동 계산.
  • 입력칸에서 수식(예: 120000+34000) 사용 가능.
  • + 버튼으로 비용 항목 추가 가능.

4.3 항공/요금/포함불포함

  • 항공 스케줄과 요금 안내는 섹션별로 행/그룹 추가 가능.
  • 각 섹션의 코드 복사로 고객 안내용 HTML 복사 가능.
  • 포함/불포함은 DB 템플릿 불러오기 및 직접 수정 지원.

5. 통합 모듈 사용법

5.1 호텔카드 메이커

  • 견적 그룹 내에서 호텔 탭 추가/편집/삭제 가능.
  • 섹션 상단 원본 사이트 버튼으로 외부 도구 열기.

5.2 새 여행 일정표

  • DAY 추가/삭제, 일정 카드 추가/편집/복제/삭제, 드래그 정렬 지원.
  • 일정표 제목은 제목 텍스트를 클릭하면 즉시 수정 모드로 전환.
  • 제목/날짜 편집 진입 시 입력칸이 자동 포커스되어 바로 입력 가능.
  • 섹션 상단 원본 사이트 버튼으로 외부 일정표 도구 열기.

5.3 url 단축기

  • 우측 상단 카드에 URL 단축기 화면이 임베드되어 있음.
  • 필요 시 새 창에서 열기 링크로 단축기만 별도 사용 가능.

6. 단축키

  • F2: 저장
  • F3: 다른 이름으로 저장
  • F4: 불러오기
  • Shift + Y: 최근 파일
  • Shift + N: 새 창

7. 문제 해결

  • 탭 전환 후 화면이 이상할 때: Ctrl+F5로 강력 새로고침.
  • 이전 화면이 계속 보일 때: 브라우저 캐시/서비스워커를 비우고 재접속.
  • 불러오기/저장이 안 될 때: 브라우저 파일 권한 요청을 허용했는지 확인.
